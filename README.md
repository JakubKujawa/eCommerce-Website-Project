# eCommerce-Website-Project

Projekt sklepu internetowego związany z przedmiotem `Zaawansowane Programowanie Obiektowe` wykonany w `Django + Vue.js`.

- [x] Zamówienia
- [ ] Obsługa płatności Stripe (Zaimplementowano, występuje problem)
